Listado de selectores CSS

  1. Type selector: selecciona elementos por su tipo
  2. ID Selector: selecciona elementos con un ID
  3. Descendant Selector: seleciona un elemento dentro de otro elemento
  4. Class Selector: seleciona elementos por su clase
  5. Comma Combinator: combina selectores con coma
  6. Selector Universal: seleccionas todo
  7. Adjacent Sibling Selector: selecciona un elemento que siga directamente a otro elemento
  8. General Sibling Selector: selecciona elementos que siguen a otro elemento
  9. Child Selector: selecciona directamente hijos de un elemento
  10. First Child Pseudo-selector: selecciona el primer elemento hijo dentro de otro elemento
  11. Only Child Pseudo-selector: seleciona un elemento que sea el único elemento dentro de otro.
  12. Last Child Pseudo-selector: seleciona el último elemento dentro de otro elemento
  13. Nth Child Pseudo-selector: selecciona un elemento por su orden en otro elemento
  14. Nth Last Child Selector: selecciona un elemento por su orden en otro elemento, contando desde atrás
  15. First of Type Selector: selecciona el primer elemento de un tipo específico
  16. Nth of Type Selector: Selecciona un elemento específico según su tipo y orden en otro elemento, o instancias pares o impares de ese elemento.
  17. Nth-of-type Selector with Formula: :nth-of-type(An+B). Selecciona cada elemento nth y comienza el recuento en una instancia específica de ese elemento
  18. Only of type Selector: selecciona elementos que sean los únicos de su tipo dentro de su elemento padre
  19. Last of Type Selector: selecciona el último elemento de un tipo específico
  20. Empty Selector: selecciona elementos que no tienen hijos
  21. Negation Pseudo-class: selecciona todos los elementos que no coincidan con el selector de negación
  22. Attribute Selector: seleciona todos los elementos que tienen atributo específico
  23. Attribute Value Selector: selecciona todos los elementos que tienen un valor de atributo específico
  24. Attribure Starts With Selector: selecciona todos los elementos con un valo rde atributo que comience con caracteres específicos
  25. Attribute Ends With Selector: selecciona todos los elementos con un valor de atributo que termine con caracteres específicos
  26. Attribute Wildcard Selector: selecciona todos los elementos con un valor de atributo que contenga caracteres específicos en cualquier lugar
